# Pixel Platformer Game

An in-progress Unity pixel platformer game project inspired by the idea of a small "Black Cat Cafe" game.
I am currently using this project to practise Unity game development, especially player movement, platform collision, level design, and iterative debugging.

## Technologies
- Unity
- C#

## Current Progress
- Created a basic Unity project structure
- Added a player movement script
- Practising character control and platformer-style movement
- Exploring Unity's scene-based workflow and object behaviour

## Files Included
- `PlayMovement.cs` — player movement script
- Selected Unity project files may be included for documentation purposes

## What I Am Learning
- How to control a player character using C#
- How Unity handles objects, scenes, and components
- How movement, collision, and visual feedback work together in a platformer game
- How to organise a Unity project and use GitHub for version control

## Note
This project is still in progress. Only selected scripts and project files are included because the full Unity project contains many generated files and asset files.
